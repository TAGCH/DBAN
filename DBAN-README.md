# Process
## Create Box
1. Create Box.
2. Write `"DBAN (model) (box Number)"`on the top and front views of the box.
3. Update the box status in [DBAN-BOXandQuantity.xlsx](https://agoda.sharepoint.com/:x:/r/sites/IT-Support/_layouts/15/Doc.aspx?sourcedoc=%7BBFAE1056-B1C4-45D3-B0A9-3C987B24907E%7D&file=DBAN-BOXandQuantity.xlsx&action=default&mobileredirect=true) to "Preparing" following the box number.

![Prepare-Status](/images/Prepare-Status.png)


## Box Full
1. Update the box status in [DBAN-BOXandQuantity.xlsx](https://agoda.sharepoint.com/:x:/r/sites/IT-Support/_layouts/15/Doc.aspx?sourcedoc=%7BBFAE1056-B1C4-45D3-B0A9-3C987B24907E%7D&file=DBAN-BOXandQuantity.xlsx&action=default&mobileredirect=true) to "Full" following the box number.

## Seal
#### **If (DBAN-*Model* following the model already exists)**
- In [SharePoint](https://agoda.sharepoint.com/sites/IT-Support/Shared%20Documents/Forms/AllItems.aspx?id=%2Fsites%2FIT%2DSupport%2FShared%20Documents%2FBKK%20Tracking&viewid=a10ac1a4%2D1e96%2D468f%2D93cc%2D22f6746deabd)
1. Create a new worksheet in DBAN-*Model* following the model by make a copy from the previous one.
2. Rename the worksheet name to "(Model)-Box(Box Number)".
![Worksheet](/images/Worksheet-Name.png)
3. Change the table head to "(Model)-Box(Box Number)"
![Table-Head](/images/Table-Head.png)
4. Collect all of serial number of laptop in the box to the table.
![Quantity](/images/Quantity.png)
5. Print out the table.
6. Stick it on the top of the box.
7. Seal the box.
8. Update the [DBAN-BOXandQuantity.xlsx](https://agoda.sharepoint.com/:x:/r/sites/IT-Support/_layouts/15/Doc.aspx?sourcedoc=%7BBFAE1056-B1C4-45D3-B0A9-3C987B24907E%7D&file=DBAN-BOXandQuantity.xlsx&action=default&mobileredirect=true) in the Quantity table following the quantity.
9. Update the box status in [DBAN-BOXandQuantity.xlsx](https://agoda.sharepoint.com/:x:/r/sites/IT-Support/_layouts/15/Doc.aspx?sourcedoc=%7BBFAE1056-B1C4-45D3-B0A9-3C987B24907E%7D&file=DBAN-BOXandQuantity.xlsx&action=default&mobileredirect=true) to "Seal" following the box number.
10. Remark on the front view of the box, "Seal".

#### **Else If (DBAN-*Model* following the model does not exists)**
- In [SharePoint](https://agoda.sharepoint.com/sites/IT-Support/Shared%20Documents/Forms/AllItems.aspx?id=%2Fsites%2FIT%2DSupport%2FShared%20Documents%2FBKK%20Tracking&viewid=a10ac1a4%2D1e96%2D468f%2D93cc%2D22f6746deabd)
1. Create a new Excel file by making a copy of the existing one then rename to "DBAN-(Model)".
2. Create a new worksheet in DBAN-*Model*.
3. Rename the worksheet name to "(Model)-Box(Box Number)".
![Worksheet](/images/Worksheet-Name.png)
4. Change the table head to "(Model)-Box(Box Number)"
![Table-Head](/images/Table-Head.png)
5. Collect all of serial number of laptop in the box to the table.
![Quantity](/images/Quantity.png)
6. Print out the table.
7. Stick it on the top of the box.
8. Seal the box.
9. Update the [DBAN-BOXandQuantity.xlsx](https://agoda.sharepoint.com/:x:/r/sites/IT-Support/_layouts/15/Doc.aspx?sourcedoc=%7BBFAE1056-B1C4-45D3-B0A9-3C987B24907E%7D&file=DBAN-BOXandQuantity.xlsx&action=default&mobileredirect=true) in the Quantity table following the quantity.
10. Update the box status in [DBAN-BOXandQuantity.xlsx](https://agoda.sharepoint.com/:x:/r/sites/IT-Support/_layouts/15/Doc.aspx?sourcedoc=%7BBFAE1056-B1C4-45D3-B0A9-3C987B24907E%7D&file=DBAN-BOXandQuantity.xlsx&action=default&mobileredirect=true) to "Seal" following the box number.
11. Remark on the front view of the box, "Seal".

## B2
1. Bring all of the seal boxes to B2 storage.
2. Update the box status in [DBAN-BOXandQuantity.xlsx](https://agoda.sharepoint.com/:x:/r/sites/IT-Support/_layouts/15/Doc.aspx?sourcedoc=%7BBFAE1056-B1C4-45D3-B0A9-3C987B24907E%7D&file=DBAN-BOXandQuantity.xlsx&action=default&mobileredirect=true) to "B2" following the box number.

`Keep every laptop before sorting.`
